1. **余数**：在 JavaScript 中，取余运算符（%）用于计算一个数除以另一个数的余数。

```
13 % 5  // 3
1 % -2 // 1
1 % 2  // 1
2 % 3  // 2
5.5 % 2 // 1.5
```
